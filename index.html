Sobre o Projeto

Este projeto tem como objetivo demonstrar o uso de HTML, CSS e JavaScript para construção de uma página interativa e dinâmica.

Foi implementado um gerador de cartões de perfil, onde o visitante pode preencher um formulário e criar dinamicamente novos integrantes na tela, simulando uma funcionalidade de painel administrativo.

Tecnologias utilizadas: HTML5, CSS3, JavaScript
